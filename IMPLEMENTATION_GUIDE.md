# Nyumba App Implementation Guide

## Architecture
This application follows a Go-inspired architecture ported to TypeScript for high-performance web rendering.

### Directory Structure
- `/handlers`: Contains API request handlers (logic).
- `/models`: Defines data structures and types.
- `/templates`: Contains the UI components (React).
- `/server`: Database configuration and initialization.
- `/main.ts`: The main entry point for the server.

## Features
- House listing and search.
- User authentication (Login/Register).
- Detailed property views.
- SQLite database persistence.

## Development
Run `npm run dev` to start the full-stack server.
