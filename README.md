# 🏠 Nyumba App

A full-stack property management application built with **Go (Golang)** and hosted on **Render**.

## 🚀 Features

* **User Roles:** Separate dashboards for Landlords and Tenants.
* **Listings:** Landlords can upload properties with **Multiple Photos**, Prices, and Categories (Bedsitter, 1BR, etc.).
* **Payments:** Integrated with **Safaricom M-Pesa (Daraja API)** for STK Push payments.
* **Communication:** Direct **WhatsApp** integration for tenant inquiries.
* **Management:** Landlords can track who booked a house and delete listings.

## 🛠️ Tech Stack

* **Backend:** Go (Golang) standard library.
* **Frontend:** Vanilla HTML/CSS/JS (Single Page Application feel).
* **Database:** JSON-based persistence (File System).
* **Hosting:** Render (Cloud).
